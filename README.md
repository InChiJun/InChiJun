### **SECS**

**주제**: 스마트 에너지케어 솔루션

**기간**: 2023.03. ~ 2023.06.

**내용**

Raspberry Pi로 환경의 온도를 제어하고, 물체가 감지될 때에만 에너지가 소비되도록 설계하였습니다.

온습도 센서로 온도를 측정하고, 특정 온도 이하로 하락하면 쿨링팬으로 온도를 제어합니다.

- 물체가 감지되지 않으면 쿨링팬은 작동하지 않습니다.

사용자는 물체감지 여부와 현재 온도를 웹 화면으로 확인할 수 있습니다.

**활용 기기**

- Raspberry Pi
- IR 적외선 센서
- 온습도 센서
- 쿨링팬

**사용 기술**

- Back-end: Python-Django
- Front-end: HTML-CSS
- DB: MySQL

**맡은 역할**

- Front-end
- Raspberry Pi 구축
